This ToDo Application is a basic task managing app, generally used to maintain our day-to-day tasks or list everything that we have to do, with the most important tasks at the top of the list, and the least important tasks at the bottom. It includes login features, where user can create their own individual ToDo App account and manage their tasks on their own. The user can create account using sign up page and login through sign in page. It also features a Contact us page, where the user can send any query to admin if they have with the application.
